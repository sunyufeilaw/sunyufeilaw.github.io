# 怎么发一篇新文章（孙律师专用说明）

这份说明只给你自己看，不会出现在网站上。

---

## 一、网站的分类长什么样

导航栏分三大栏目：

- **成功案例**（鼠标移上去会弹出 6 个领域）
  - 劳动争议 / 工伤 / 建设工程 / 民间借贷 / 合同纠纷 / 交通事故
- **法律科普**
- **生活随笔**
- **全部文章**（所有文章按时间排）

文章归到哪一栏，**完全由你给它贴的「标签（tags）」决定**，不用手动建页面。

---

## 二、贴标签规则（最重要，记住这一条）

| 你写的是…… | 标签 `tags` 就填 |
|---|---|
| 一个办过的案子 | 对应领域名，如 `[劳动争议]`、`[工伤]`、`[建设工程]`、`[民间借贷]`、`[合同纠纷]`、`[交通事故]` |
| 给普通人的法律科普 | `[法律科普]` |
| 生活感悟、随笔 | `[生活随笔]` |

一句话：**案例填领域名，科普填「法律科普」，随笔填「生活随笔」。** 三类绝不串台。

---

## 三、发一篇新文章的步骤

1. 打开 `_posts` 文件夹，**复制**一篇已有的文章（案例就复制那篇案例样板）。
2. 把复制出来的文件**改名**，格式是「日期-英文标题」，例如 `2026-07-01-gongshang-case.md`。
   - 前面的日期必须是 `年-月-日` 这种写法，否则文章不显示。
3. 用记事本或编辑器打开，改最上面 `---` 之间这几行：
   ```
   title: 这里写文章标题
   subtitle: 这里写副标题（可留空）
   tags: [劳动争议]      ← 按上面的规则填
   ```
4. 在下面正文区域写内容（写法见第四节）。
5. 保存，推送到 GitHub，过 1～2 分钟刷新网站就能看到。

---

## 四、正文怎么写（Markdown 简易规则）

- `## 小标题`：在文字前加两个井号和一个空格
- `**加粗**`：两边各加两个星号
- `- 列表项`：行首一个减号加空格
- 插入图片（如打码判决书）：
  1. 先把图片放进 `assets/img/` 文件夹；
  2. 在正文里写一行：`![判决书](/assets/img/图片文件名.png)`

---

## 五、上传判决书的安全提醒 ⚠️

附判决书/裁决书前**务必打码**，以下都要遮挡：

- 当事人姓名、身份证号、手机号、住址
- 公司全称、银行账号、案号中可识别身份的部分

建议只截取「本院认为」和判决结果部分，既能证明实力，又不泄露当事人隐私。

---

## 六、想增加 / 修改分类怎么办

打开 `_config.yml`，找到 `navbar-links` 那一段，照着已有的格式加一行即可，例如想加「婚姻家庭」：

```
    - 婚姻家庭: "/tags/#婚姻家庭"
```

然后给对应文章贴上 `tags: [婚姻家庭]` 就行。改完保存、推送即可。
