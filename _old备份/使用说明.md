# 个人网站 · 使用说明

这是给你自己看的操作手册。不用懂技术,照着做就行。

---

## 一、网站是怎么组成的

整个网站由 4 个文件分工合作,各管各的:

```
sunyufeilaw.github.io/
├─ index.html          网页骨架(不用动)
├─ css/style.css       ⭐ 改风格(颜色 / 字体 / 排版)来这里
└─ js/
   ├─ content.js       ⭐ 改文字 / 加作品来这里(你最常用)
   └─ main.js          显示逻辑(永远不用动)
```

一句话记住:**改文字找 `content.js`,改长相找 `style.css`,其它别碰。**

---

## 二、怎么预览效果(改完怎么看)

1. 打开网站文件夹。
2. **双击 `index.html`**,电脑会用浏览器打开,这就是你网站的样子。
3. 改完文件并保存后,回到浏览器**按 F5 刷新**,就能看到最新效果。

---

## 三、怎么改文字

1. 用记事本或编辑器打开 `js/content.js`。
2. 找到你要改的地方,**只改引号 `"` 里面的字**。
3. 保存,刷新浏览器查看。

> ⚠️ 唯一的规矩:**别删掉引号 `"` 和逗号 `,`**。只换引号里的字,就不会出错。

---

## 四、怎么加一个新作品

打开 `js/content.js`,找到 `works`(作品)这一段:

```js
works: [
  { 标题: "执行异议之诉", 介绍: "简单介绍这个作品或经历。" },
  { 标题: "作品二",       介绍: "再写一个,没有就先留着。" },
],
```

复制其中一行,粘到下面,改成你的内容(**结尾的逗号别丢**):

```js
works: [
  { 标题: "执行异议之诉", 介绍: "简单介绍这个作品或经历。" },
  { 标题: "作品二",       介绍: "再写一个,没有就先留着。" },
  { 标题: "我的新作品",   介绍: "这里写一句话介绍。" },
],
```

保存、刷新,新卡片就出现了。删作品就把整行删掉。

---

## 五、怎么改风格(颜色 / 字体)

打开 `css/style.css`,里面每段都有中文注释告诉你管什么。
不确定的话,直接把你的想法告诉 AI 助手(比如"想要稳重的黑白风格"),让它帮你改这个文件就行,**改风格不会影响你的文字内容**。

---

## 六、怎么把修改发布到网上(GitHub Pages)

改完保存后,网上的网站不会自动更新,需要把改动"上传"一次:

- 如果你用的是 **GitHub Desktop**(图形软件):打开它 → 左边会列出你改了哪些文件 → 在下方填一句话(比如"更新作品")→ 点 **Commit** → 再点 **Push**。
- 等一两分钟,刷新你的网址就能看到更新。

> 网址通常是:`https://sunyufeilaw.github.io`

---

## 七、万一改坏了怎么办

别慌,有两种办法:

1. 把刚才改动的地方改回去(常见是漏了引号或逗号)。
2. 如果用 GitHub Desktop,可以右键那个文件选 **Discard changes(放弃修改)**,就回到上次正常的样子。

实在搞不定,把情况告诉 AI 助手,帮你修。
